# Call-Center-Performance-Analysis
This project presents an interactive Tableau dashboard designed to analyze call center operations and customer experience. It focuses on key performance metrics such as call volume, response time, customer satisfaction (CSAT), sentiment analysis, and communication channels.
The dataset contains over 32,000 records, allowing a comprehensive analysis of call center performance across multiple locations. The dashboard helps identify customer behavior patterns, operational efficiency, and areas that require improvement to enhance overall service quality.
Key Features
📞 Analysis of total call volume across call centers
😊 Customer sentiment and satisfaction evaluation
⏱️ Response time and SLA performance tracking
📊 Channel-wise distribution (Call, Email, Chatbot, Web)
🌍 Location-based performance comparison
Key Insights
Certain locations handle a higher volume of calls
Billing-related queries are the most common
Negative customer sentiment is relatively high
Most calls are resolved within SLA
Tech Stack
Tableau (Dashboard & Data Visualization)
Data Analysis & KPI Metrics
Use Case
This project demonstrates how data visualization can support data-driven decision-making in customer service operations. It helps organizations improve response efficiency, enhance customer satisfaction, and optimize resource allocation.
